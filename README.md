# RideGuard - Organized by User Role

This version separates the app prototype by user role.

## Files

- `index.html` - landing page with links to each view
- `parent/index.html` - Parent screens only
- `driver/index.html` - Driver screens only
- `guard/index.html` - Guard screens only
- `shared/css/styles.css` - shared styling
- `shared/js/app.js` - shared JavaScript interactions

Open `index.html` first, or open any role folder directly.
